Elasticsearch: Motor de búsqueda es una buena base para generar una plataforma funcional 
InfluxDB: Una base de datos de series temporales (optimizada para datos que cambian con el tiempo).
	Ideal para monitoreo, métricas, eventos y cualquier dato con sello de tiempo. 

Grafana: Una plataforma de visualización de métricas (gráficos, dashboards, alertas).
	Se usa universalmente para observar datos de series temporales de muchas fuentes. 

Conector de InfluxDB en Grafana:Grafana tiene que “conectarse” a InfluxDB para poder leer los datos y visualizarlos.
	El video muestra cómo configurar ese data source de forma que Grafana pueda consultar InfluxDB.
